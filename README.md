Login & Registration Form – Project Description
 This project is a simple and responsive Login and Registration Form built using HTML, CSS, and JavaScript. 
 It allows users to create an account by registering their details and then log in using their credentials.
 The design is clean, user-friendly, and suitable for beginners learning frontend development.
 
Features:
  User-friendly and responsive UI
  Login and Signup pages
  Form validation using JavaScript
  Error messages for invalid input
  Clean and modern design
 Easy to integrate into any website
 
 Tech Stack:
HTML5 – Structure
CSS3 – Styling and layout
JavaScript – Form validation and dynamic interactions

Folder Structure:
/project-folder
│── index.html
│── login.html
│── register.html
│── style.css
│── script.js
└── assets/

 How It Works:
*Users enter details such as username, email, and password.
*JavaScript checks if the input is valid.
*If valid → the user can log in using their email and password.
*Incorrect inputs show validation errors.
